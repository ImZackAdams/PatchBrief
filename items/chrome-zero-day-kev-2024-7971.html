CISA KEV adds actively exploited Chrome V8 type confusion zero-day

Date Aug 21, 2024
Type KEV
Signal Known exploited
Vendor / Product Google · Chrome (desktop)
CVE CVE-2024-7971
Known exploited CVE-2024-7971

Summary

Google confirmed exploitation in the wild for CVE-2024-7971, a type confusion vulnerability in the V8 JavaScript engine. CISA added it to the KEV catalog. All desktop Chrome versions before 128.0.6613.84 are affected.

Why it matters

Zero-day with active exploitation and a CISA KEV addition is a strong signal. Browser vulnerabilities affect every endpoint in a fleet. Unmanaged or infrequently updated endpoints are the primary risk surface.

Operator check

Check browser fleet versions against Chrome 128.0.6613.84 or later. Verify update policy for laptops frequently off-network. Flag unmanaged browsers for manual follow-up.
