Aug 21, 2024 KEV Known exploited CVE-2024-7971

CISA KEV adds actively exploited Chrome V8 type confusion zero-day

Google · Chrome (desktop)

Google confirmed exploitation in the wild for CVE-2024-7971, a type confusion vulnerability in the V8 JavaScript engine. CISA added it to the KEV catalog. All desktop Chrome versions before 128.0.6613.84 are affected.

Feb 14, 2024 Ransomware Threat activity

ALPHV/BlackCat ransomware targets healthcare sector via unpatched VPN appliances

Multiple · VPN appliances (Citrix, Cisco, Fortinet)

CISA and the FBI issued a joint advisory on ALPHV/BlackCat ransomware following healthcare sector intrusions through unpatched VPN appliances. The group has exploited known vulnerabilities in Citrix NetScaler, Cisco ASA, and Fortinet devices as initial access vectors.

Ivanti Connect Secure authentication bypass and command injection — active exploitation

Ivanti · Connect Secure, Policy Secure

Oct 16, 2023 KEV Known exploited CVE-2023-20198

CISA KEV: Cisco IOS XE Web UI privilege escalation — zero-day mass exploitation

Cisco · IOS XE

Cisco disclosed CVE-2023-20198, a critical zero-day in IOS XE allowing unauthenticated attackers to create a privilege level 15 account via the Web UI feature. Mass exploitation was observed within hours of disclosure, with thousands of devices implanted with a backdoor.

Operator check

Disable the HTTP Server feature on all internet-facing IOS XE systems immediately. Check for unauthorized accounts and the presence of the implant as described in Cisco's advisory.
