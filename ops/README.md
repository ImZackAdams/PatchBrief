# Ops

This directory contains pilot operations templates and tooling.

## Pilot tracker

Copy `pilot-tracker-template.csv` to `pilot-tracker.csv` before use.

**Do not commit `pilot-tracker.csv` or any file with real user data.** It is gitignored.

Customer/user data files belong in `ops/customer-data/` (also gitignored).

## Email templates

Reusable Markdown templates for pilot communication are in `ops/email-templates/`.

Fill in `{{placeholder}}` values before sending.
